# maya-SkinWeight-Transfer-Tool
Mayaでスキンウェイトを複製モデルに自動転送するツール

## ■ 概要
Maya上でスキンウェイトを他のモデルへ自動で転送するツールです。

複製したモデルに対して手動でウェイトを設定する手間を削減し、作業効率の向上を目的として作成しました。

---

## ■ 機能
・スキンウェイトのコピー  
・複製モデルへの自動適用  
・選択モデルをもとに処理実行  

---

## ■ 使用技術
・Python  
・maya.cmds  

---

## ■ 使い方
1. 転送元（元のウェイトを持つモデル）を選択  
2. 転送先（適用したいモデル）を選択  
3. スクリプトを実行  
4. ウェイトが自動で転送される  

---

## ■ 背景・目的
キャラクターモデルや複製モデルに対してウェイトを設定する際、
毎回手作業で調整する必要があり時間がかかっていました。

そのため、ウェイト設定作業を自動化し、効率化することを目的として制作しました。

---

## ■ 工夫した点
・Maya内での選択オブジェクトを取得し、自動処理に組み込んだ  
・一連の工程をスクリプト化して作業時間を短縮
